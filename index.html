Quick Start

From zero to your first API call in 30 seconds.

1

Get your free API key

No signup form, no email verification. One POST request.

bash
curl -X POST https://api-catalog-three.vercel.app/api/keys/create
2

Make your first API call

Look up any IP address, get geolocation data back in JSON.

bash
curl -H "Authorization: Bearer YOUR_KEY" \
  https://api-catalog-three.vercel.app/v1/agent-geo/geo/8.8.8.8
3

Response

json
{
  "ip": "8.8.8.8",
  "city": "Mountain View",
  "region": "California",
  "country": "US",
  "loc": "37.3861,-122.0839",
  "org": "AS15169 Google LLC",
  "timezone": "America/Los_Angeles"
}
