API Endpoints

  
Two endpoints for different use cases


  

    GET
    /v1/defi-trading/prices
    
Simple price feed — 500+ tokens, returns symbol/price pairs. Best for dashboards, price displays, and quick lookups.

  


  

    GET
    /v1/crypto-feeds/api/prices
    
Detailed market data — 38 major tokens with OHLCV data: open, high, low, volume, 24h change percent. Best for trading bots and analytics.

  


  

    GET
    /v1/crypto-feeds/api/price/{SYMBOL}
    
Single token lookup — get detailed price data for one specific token. Example: /v1/crypto-feeds/api/price/BTC

Frequently Asked Questions

  

    
Is the crypto price API really free?

    
Yes. You get 200 free API credits instantly with no signup, no credit card, and no email required. Each API call costs 1 credit. After the free tier, credits cost $1 per 500 calls ($0.002 per request).

  

  

    
How many cryptocurrencies does the API support?

    
The simple endpoint covers 500+ tokens including Bitcoin (BTC), Ethereum (ETH), Solana (SOL), BNB, XRP, Dogecoin, and hundreds of altcoins and DeFi tokens. The detailed endpoint provides OHLCV data for 38 major tokens. Prices are sourced from major exchanges in real-time.

  

  

    
How do I get an API key?

    
Just run one command:
curl -X POST https://agent-gateway-kappa.vercel.app/api/keys/create
You'll get an API key back instantly. No account, no email, no signup form. Start making requests immediately.

  

  

    
What response format does the API use?

    
All responses are JSON. The simple endpoint returns symbol/price pairs: {"BTC": "72257.50", "ETH": "2113.45", ...}. The detailed endpoint returns OHLCV objects with open, high, low, volume, 24h change, and percentage change for each token.

  

  

    
How does this compare to CoinGecko?

    
CoinGecko requires email signup, has aggressive rate limits (30 calls/min on free tier), and recently made their free tier more restrictive. Frostbyte gives you instant access with no signup and a faster API. Plus, the same API key also works for 40+ other services (IP geolocation, screenshots, DNS lookup, web scraping, and more).

  

  

    
Can I use this API in production?

    
Yes. The API runs on production infrastructure with sub-200ms response times and consistent JSON responses. For higher volume, top up credits with USDC on Base blockchain ($1 = 500 credits). Uptime is monitored 24/7.

  

  

    
What else can I do with the API key?

    
Your API key works across 40+ services: IP geolocation, website screenshots, DNS lookup, WHOIS, web scraping, code execution, PDF generation, URL shortening, web search, and more. One key for everything. See the full API catalog.
