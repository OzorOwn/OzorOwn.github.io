Frequently Asked Questions

  

    
Is the IP geolocation API free?

    
Yes. You get 200 free API credits instantly. No signup, no credit card, no email. Each lookup costs 1 credit. After the free tier, credits cost $1 per 500 lookups.

  

  

    
What data does the API return?

    
Country, country code, region, city, latitude, longitude, timezone, ISP, organization, and AS number. All returned as a single JSON object.

  

  

    
How accurate is the geolocation?

    
Country-level accuracy is 99%+. City-level accuracy is approximately 80-90% for most regions. Data is sourced from multiple geolocation databases and updated regularly.

  

  

    
How does this compare to MaxMind?

    
MaxMind requires account creation, downloading databases, and local infrastructure setup. Frostbyte is a simple REST API — one HTTP request returns geolocation data. No database downloads, no account, no server-side setup needed.

  

  

    
Can I look up my own IP?

    
Yes. The live demo above auto-detects your IP. Via the API, you can pass any IP address to get its location data.

  

  

    
What else can I do with the API key?

    
Your API key works across 40+ services: crypto prices, website screenshots, DNS lookup, WHOIS, web scraping, code execution, PDF generation, and more. One key for everything. See the full API catalog.
